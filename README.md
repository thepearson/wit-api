wit-api
=======

Web image tools API

api params as follows:

key=[USER UUID]
	eg:	key=fcc42e8e-11ca-11e2-83d9-63d8f8d29f08

u=[Site URL]
	eg: u=http://www.google.com

width=[Browser window width]
	eg: width=1024

height=[Browser window height]
	eg: height=768

resize=[percentage to resize eng image by]
	eg: resize=50
